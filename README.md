**Anispectra** — bu Next.js 14 (App Router) yordamida yozilgan, yuqori darajada optimallashtirilgan, to'liq bepul ma'lumot manbalariga asoslangan premium multimedia portalidir. Foydalanuvchilar bu yerda eng so'nggi anime, dorama va mangalarni tomosha qilishlari, o'qishlari va o'zlarining ko'rish/o'qish tarixlari (history)ni saqlab borishlari mumkin.

## 🛠️ Ishlatilgan Texnologiyalar & Kutubxonalar

* **Framework:** Next.js 14 (App Router)
* **Til:** TypeScript
* **Dizayn & Stil:** Tailwind CSS, Lucide React (Ikonlar)
* **Ma'lumotlar Bazasi & ORM:** Prisma ORM, `@prisma/adapter-libsql` (SQLite va LibSQL bilan)
* **Autentifikatsiya:** NextAuth.js (Google Provider & OTP support)
* **Video Player:** `hls.js` (HTTP Live Streaming m3u8 oqimlarini o'ynatish uchun)

## 🏗️ Loyiha Strukturasi

```bash
src/
 ├── app/
 │    ├── anime/         # Anime katalogi va HLS pleer sahifasi
 │    ├── dorama/        # Dorama katalogi va Kodik iframe pleer sahifasi
 │    ├── manga/         # Manga katalogi va boblar o'quvchi sahifa (Reader)
 │    ├── auth/          # Google Login & OTP orqali kirish sahifalari
 │    ├── profile/       # Foydalanuvchi profili va ko'rishlar tarixi (History)
 │    ├── search/        # Barcha bo'limlar bo'yicha global qidiruv
 │    ├── collections/   # Foydalanuvchi kolleksiyalari
 │    ├── layout.tsx     # Bosh sahifa tuzilishi va provider-lar
 │    └── globals.css    # Maxsus Tailwind va animatsiya stillari
 ├── components/         # UI komponentlar (Player, Header, Footer, PosterCard va boshqalar)
 ├── lib/                # API integratsiya xizmatlari (Anime, Manga, Kodik API)
 ├── server/             # Prisma DB mijoz va server xizmatlari (History Service)
 └── types/              # TypeScript turlari va NextAuth interfeyslari
```

## 💻 Mahalliy Kompyuterda Ishga Tushirish

Loyiha mahalliy kompyuterda faqat **localhost:3000** portida ishlaydigan qilib sozlangan.

### 1. Loyihani yuklab oling va bog'liqliklarni o'rnating:
```bash
npm install
```

### 2. Environment (muhit) fayllarini sozlang:
`.env.example` faylidan nusxa olib, `.env.local` faylini yarating va kerakli kalitlarni kiriting:
```bash
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=sizning_nextauth_maxfiy_kalitingiz
DATABASE_URL=file:./dev.db

# Kodik Dorama tokeni
KODIK_API_TOKEN=sizning_kodik_tokeningiz

# Google OAuth hisoblari (Google login uchun)
GOOGLE_CLIENT_ID=google_client_id
GOOGLE_CLIENT_SECRET=google_client_secret
```

### 3. Avtomatik tozalash va localhost:3000 da ishga tushirish:
Loyihani avvalgi keshlaridan tozalab, to'g'ridan-to'g'ri `3000`-portda xavfsiz ishga tushirish uchun maxsus yozilgan skriptni ishlating:
```bash
node clean_and_start.js
```
Bu skript avtomatik ravishda:
1. `3000`-portni band qilib turgan har qanday eski jarayonni yopadi (o'chiradi).
2. `.next` papkasidagi eski keshlarni o'chiradi.
3. Loyihani faqat `localhost:3000` manzilida ishga tushiradi.

## 🛡️ Xavfsizlik & Tarmoq Barqarorligi

* **Kodik API Token Himoyasi:** Kodik API tokenlari faqat server tomonda (`server-side`) ishlatiladi va brauzerga hech qachon sizib chiqmaydi.
* **Tizim Barqarorligi (Graceful Degradation):** Tashqi API xizmatlaridan biri (masalan, MangaDex yoki AniLiberty) vaqtincha ishlamay qolganda, tizim butunlay qulab tushmaydi. Maxsus `EmptyState` komponenti foydalanuvchiga muammoni chiroyli tarzda tushuntiradi.
